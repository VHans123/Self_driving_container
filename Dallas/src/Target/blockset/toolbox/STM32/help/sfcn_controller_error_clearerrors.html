Clear errors from controller

Block to clear errors from the RAM or EEPROM of the controller

Description

All active/stored errors will be erased from the RAM/EEPROM.

Remarks

  • Active errors are saved in RAM, they are lost by rebooting the ECU or by clearing the active error list.
  • Stored errors are programmed in EEPROM, they are only lost by clearing the stored error list.
  • Error codes 0 to 0x8000 are reserved for internal system errors.
Declaration

(No initialisation block needed)
