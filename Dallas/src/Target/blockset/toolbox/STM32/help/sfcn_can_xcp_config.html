XCP over CAN config

Block to use and config the XCP over CAN protocol

Description

A 'XCP Config' block instructs the code generator to include the XCP over CAN protocol.

Remarks

  • Only one instance of the XCP protocol is currently allowed on the controller.
Declaration

Use the CAN config block:

  • To set the baudrate for the selected CAN channel.
Parameters
CAN channel
Select the used channel.
CRO identifier
Specify the CAN CRO message identifier for XCP (master->slave).
DTO identifier
Specify the CAN DTO message identifier for XCP (slave->master).
