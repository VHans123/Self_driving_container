SPI master init

Block to initialize a SPI master channel

Description

The block initializes a SPI master channel with the communication settings as specified by the parameters.

Parameters
Channel
Select the SPI channel
Communication speed
Select the frequency of the SPI clock signal.
Polarity
Select the polarity of the SPI clock signal during idle.
Phase
Select the edge of the SPI clock signal at which time a bit should be sampled.
