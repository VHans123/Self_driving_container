Get Free Running Counter

Reads the free running counter.

Description

Each timer module has an internal 16-bit free running counter that continuously counts from 0 to 65535. The speed at which the counter runs is determine by the frequency that the timer modules was initialized for. The 'Get Free Running Counter' block reads the current value of this 16-bit free running counter.

Output

16-bit free running counter value.
The used datatype is of uint16.

Parameters
Timer module
Selects the timer module.
Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model.
