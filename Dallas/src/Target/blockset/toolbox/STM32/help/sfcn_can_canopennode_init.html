CANopenNode Init

Block to initialize the CANopenNode stack for a particular CAN channel.

Description

A 'CANopenNode Init' block instructs the code generator to include the CANopen communication stack. Additionally, it specified the CAN channel to use for the CANopen communication.

Remarks

  • Only one instance of the CANopen communication stack is currently allowed on the controller.
  • The baudrate setting in object dictionary 2102h (125 kbit/sec) overrides the value specified in the 'CAN config' block.
Parameters
CAN channel
Select the used channel to be used by the CANopen communication stack.
